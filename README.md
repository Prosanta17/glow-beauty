# Glow Beauty ✨

A modern **Beauty Products E-commerce UI** built with **React, Vite, Tailwind CSS, and Redux Toolkit**.
The project demonstrates a scalable frontend architecture with modern UI libraries and state management.

---

## 🚀 Tech Stack

- **React 19**
- **Vite 7**
- **TypeScript**
- **Tailwind CSS**
- **Redux Toolkit**
- **React Router DOM**
- **TanStack React Query**
- **Radix UI**
- **Framer Motion**
- **Lucide Icons**

---

## 🎨 Features

- Modern **Beauty E-commerce UI**
- Responsive design (mobile-first)
- Product listing layout
- Animated UI using **Framer Motion**
- Icon system with **Lucide React**
- Global state management with **Redux Toolkit**
- Server-state handling with **React Query**
- Utility styling with **Tailwind CSS**
- Accessible UI components via **Radix UI**
- Clean and scalable component structure

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Prosanta17/glow-beauty.git
```

Go to the project directory:

```bash
cd glow-beauty
```

Install dependencies:

```bash
npm install
```

---

## 🧑‍💻 Run Development Server

```bash
npm run dev
```

The app will run on:

```
http://localhost:5173
```

---

## 📦 Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

---

## 🧹 Lint Code

```bash
npm run lint
```
